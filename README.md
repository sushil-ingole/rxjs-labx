## ⚡ What is rxjs-labx?

**rxjs-labx** is an interactive playground where you don’t just read RxJS —
you *experience* how it behaves.

Change inputs.
See outputs.
Understand instantly.

---

## 🎯 Why this exists

RxJS is powerful… but confusing.

Most resources explain concepts,
but don’t clearly show **how things behave over time**.

**rxjs-labx fixes that.**

---

## 🧪 What you can do

* 🔄 Experiment with RxJS operators in real-time
* ⏱️ Visualize timing and async behavior
* 🎛️ Modify inputs and instantly see output changes
* 🧠 Build intuition instead of memorizing
